# Thai Adblock Lists

### บทนำ

Repository นี้ถูกสร้างขึ้นมาเพื่อให้คนไทยที่ใช้งานอินเทอร์เน็ตสามารถใช้งานได้ง่ายขึ้น โดยการทำ Adblock List ที่เหมาะสมกับคนไทย และสามารถร่วมพัฒนาได้ และสามารถนำไปใช้งานได้ฟรีโดยไม่มีค่าใช้จ่ายใดๆทั้งสิ้น

### วัตถุประสงค์

จุดประสงค์หลักของพื้นที่เก็บข้อมูลนี้คือการบล็อกโฆษณาและเนื้อหาที่ไม่พึงประสงค์บนเว็บไซต์ภาษาไทยต่างๆ การใช้ AdBlock List ผู้ใช้สามารถเพลิดเพลินกับประสบการณ์การท่องเว็บที่สะอาดและคล่องตัวยิ่งขึ้นโดยไม่มีโฆษณารบกวน ประโยชน์หลักประกอบด้วย:

- บล็อกโฆษณารบกวนบนเว็บไซต์ภาษาไทย
- บล็อกโฆษณาที่เป็นอันตราย รวมถึงโฆษณาการพนันผิดกฎหมาย
- เพิ่มความเร็วในการโหลดหน้าเว็บและลดการใช้แบนด์วิดท์
- มอบประสบการณ์การท่องเว็บที่สะอาดและปลอดภัยยิ่งขึ้น

### เริ่มต้นใช้งาน

หากต้องการใช้ Thai Adblock List คุณสามารถทำตามขั้นตอนต่อไปนี้:

1. ติดตั้งส่วนขยาย Adblock บนเว็บเบราว์เซอร์ของคุณ ตัวเลือกยอดนิยมได้แก่:

   - uBlock Origin (แนะนำ)
   - Adblock Plus
   - AdGuard
   - Brave Browser (มี adblocker ในตัว)

2. เปิดใช้งานส่วนขยายตัวบล็อกโฆษณาบนเว็บเบราว์เซอร์ของคุณ

3. ไปที่การตั้งค่าหรือตัวเลือกของส่วนขยาย Adblock ที่คุณติดตั้ง และเลือกเมนู "ตัวกรอง" หรือ "ตัวกรองเพิ่มเติม" หรือ "ตัวกรองที่กำหนดเอง"

4. เพิ่ม URL ต่อไปนี้เป็นรายการ adblock ที่กำหนดเอง:

   [Thai Adblock List RAW](https://raw.githubusercontent.com/PhyschicWinter9/thai-adblock-list/main/thai-adblock-list.txt)

5. บันทึกการตั้งค่าและโหลดหน้าเว็บที่คุณเข้าชมใหม่เพื่อเริ่มเพลิดเพลินกับประสบการณ์การเรียกดูที่ไม่มีโฆษณา

### คุณสมบัติ

- ปรับแต่งมาสำหรับเว็บไซต์ภาษาไทยและเนื้อหาภาษาไทยโดยเฉพาะ
- บล็อกโฆษณาที่เป็นอันตราย รวมถึงโฆษณาการพนันผิดกฎหมาย
- เพิ่มความเร็วในการโหลดหน้าเว็บและลดการใช้แบนด์วิดท์
- อัปเดตอย่างสม่ำเสมอเพื่อรักษาประสิทธิภาพ
- ใช้งานได้กับส่วนขยายบล็อกโฆษณาหลักๆ (uBlock Origin, Adblock Plus, AdGuard)
- รองรับทั้งเบราว์เซอร์บนเดสก์ท็อปและมือถือ

### การตรวจสอบความถูกต้อง (Checksums)

เพื่อความสะดวกของคุณ คุณสามารถใช้ข้อมูลตรวจสอบความถูกต้องเหล่านี้เพื่อตรวจสอบความถูกต้องของไฟล์รายการบล็อก:

- SHA-1: cd0864e2bfbeef7ed2c02d03fd3726fe7b30e748 thai-adblock-list.txt
- MD5: 00cacef56b345aef9dfc08e7fb7a30c2 thai-adblock-list.txt

โปรดทราบว่าข้อมูลตรวจสอบความถูกต้องเป็นวิธีหนึ่งในการตรวจสอบความถูกต้องของไฟล์และให้มั่นใจว่าไฟล์ไม่ได้ถูกแก้ไข

### การมีส่วนร่วม

เราขอเชิญชวนให้ท่านมีส่วนร่วมในการพัฒนารายการบล็อกโฆษณาไทย ซึ่งได้รับการต้อนรับและชื่นชมอย่างยิ่ง ในการมีส่วนร่วม กรุณาปฏิบัติตามขั้นตอนเหล่านี้:

1. Fork ที่เก็บข้อมูลนี้ไปยังบัญชี GitHub ของคุณ

2. ทำการเพิ่มหรือแก้ไขไฟล์รายการบล็อกโฆษณาตามที่ต้องการ โดยให้แน่ใจว่ารูปแบบที่กำหนดไว้ถูกต้อง

3. ทดสอบการเปลี่ยนแปลงของคุณอย่างละเอียดเพื่อให้แน่ใจว่าทำงานได้อย่างถูกต้องและไม่ทำให้เว็บไซต์ใช้งานไม่ได้

4. ส่ง pull request พร้อมระบุการเปลี่ยนแปลงที่คุณได้ทำและเหตุผล รวมถึง:

   - ประเภทของโฆษณาหรือเนื้อหาที่ตัวกรองของคุณบล็อก
   - เหตุผลที่จำเป็นต้องเปลี่ยนแปลง
   - การทดสอบที่คุณได้ทำ

5. การมีส่วนร่วมของคุณจะได้รับการตรวจสอบ และหากได้รับการอนุมัติจะถูกรวมเข้ากับที่เก็บข้อมูลหลักเพื่อประโยชน์ของผู้ใช้ทุกคน

โปรดทราบว่าการมีส่วนร่วมทั้งหมดควรสอดคล้องกับเป้าหมายในการบล็อกโฆษณาและเนื้อหาที่ไม่พึงประสงค์สำหรับเว็บไซต์ภาษาไทย

### รูปแบบตัวกรอง (Filter Syntax)

รายการบล็อกโฆษณานี้ใช้รูปแบบตัวกรองมาตรฐานที่เข้ากันได้กับ:

- Adblock Plus syntax
- uBlock Origin extended syntax

สำหรับข้อมูลเพิ่มเติมเกี่ยวกับรูปแบบตัวกรอง โปรดดูที่:

- [Adblock Plus Filter Documentation](https://help.adblock.com/hc/en-us/articles/360062733293)
- [uBlock Origin Filter Syntax](https://github.com/gorhill/uBlock/wiki/Static-filter-syntax)

### สำหรับผู้ดูแลและผู้มีส่วนร่วม

#### การจัดการรายการเว็บไซต์

เพื่อดูแลและอัปเดตตัวกรองอย่างมีประสิทธิภาพในขณะที่รักษาที่เก็บข้อมูลให้สะอาดและถูกต้องตามกฎหมาย:

**1. ใช้ Comments ในไฟล์ตัวกรอง**

จัดระเบียบตัวกรองของคุณโดยใช้ section comments ในไฟล์ `thai-adblock-list.txt`:

```
! ===== ส่วนที่ 1: เว็บไซต์ไทยทั่วไป =====
! อัปเดตล่าสุด: 25-09-2025
[filter rules here]

! ===== ส่วนที่ 2: เว็บไซต์สตรีมมิ่ง =====
! อัปเดตล่าสุด: 25-09-2025
[filter rules here]
```

**2. เก็บบันทึกส่วนตัวแยกต่างหาก**

ติดตามเว็บไซต์เฉพาะในไฟล์ส่วนตัวของคุณ (ไม่ commit ลง GitHub):

- ใช้สเปรดชีตในเครื่อง (Excel, Google Sheets, Notion)
- สร้างเอกสารส่วนตัวในคอมพิวเตอร์ของคุณ
- เพิ่มไฟล์ติดตามส่วนตัวใน `.gitignore`:

```
# ไฟล์ติดตามส่วนตัว
INTERNAL_NOTES.md
website-tracking.xlsx
notes/
*.private.md
```

**3. แนวปฏิบัติที่ดีสำหรับการอัปเดต**

ควรทำ:

- ใช้ข้อความ commit แบบทั่วไป: "Update filter rules", "Fix popup blockers", "Improve ad blocking"
- ทดสอบการเปลี่ยนแปลงบนหลายเว็บไซต์ก่อน commit
- บันทึกประเภทของโฆษณาที่ถูกบล็อกใน filter comments
- เก็บบันทึกรายละเอียดในไฟล์ส่วนตัว

ไม่ควรทำ:

- กล่าวถึงชื่อเว็บไซต์เฉพาะใน commit messages
- Commit ไฟล์ติดตามส่วนตัวลงในที่เก็บข้อมูลสาธารณะ
- สร้างรายการสาธารณะของเว็บไซต์ที่อาจมีปัญหาทางกฎหมาย

**4. เคล็ดลับการจัดระเบียบตัวกรอง**

การใส่ comment ที่มีประสิทธิภาพช่วยในการดูแลและแก้ไขตัวกรอง:

```
! วัตถุประสงค์: บล็อกโฆษณาบนเครื่องเล่นวิดีโอ
! ประเภท: Popup, overlay, pre-roll
! ความสำคัญ: สูง
example.com##.video-ad-overlay
example.com##.popup-container

! วัตถุประสงค์: บล็อกโฆษณาการพนัน
! ประเภท: Banner, sidebar
! ความสำคัญ: วิกฤต (เนื้อหาผิดกฎหมาย)
another-site.com##.gambling-banner
```

วิธีการนี้ช่วยให้ผู้ดูแลสามารถติดตามและอัปเดตตัวกรองได้อย่างมีประสิทธิภาพในขณะที่รักษาที่เก็บข้อมูลสาธารณะให้สะอาดและปลอดภัยทางกฎหมาย

### การรายงานปัญหา

หากคุณพบปัญหาใดๆ หรือมีข้อเสนอแนะในการปรับปรุง กรุณาเปิด issue บนตัวติดตามปัญหาของที่เก็บข้อมูลนี้

#### ประเภทของปัญหาที่คุณสามารถรายงานได้:

**1. โฆษณาที่ไม่ถูกบล็อก**

- ระบุ URL ของเว็บไซต์ (หากเหมาะสม)
- อธิบายประเภทของโฆษณา (popup, banner, วิดีโอก่อนเล่น, ฯลฯ)
- แนบภาพหน้าจอแสดงโฆษณา (เบลอเนื้อหาที่ละเอียดอ่อนหากจำเป็น)

**2. ปัญหาการทำงานของเว็บไซต์**

- รายงานหากรายการบล็อกโฆษณาทำให้คุณสมบัติของเว็บไซต์ใช้งานไม่ได้
- อธิบายสิ่งที่หยุดทำงาน
- รวมขั้นตอนในการทำซ้ำปัญหา

**3. False Positives (บล็อกเนื้อหาที่ไม่ควรบล็อก)**

- รายงานหากเนื้อหาที่ถูกต้องถูกบล็อก
- ระบุองค์ประกอบใดที่ถูกบล็อกอย่างไม่ถูกต้อง

**4. ขอเพิ่มตัวกรอง**

- ขอให้บล็อกโฆษณาประเภทเฉพาะบนเว็บไซต์ภาษาไทย
- อธิบายพฤติกรรมการโฆษณาที่คุณต้องการบล็อก
- หมายเหตุ: คุณไม่จำเป็นต้องระบุชื่อเว็บไซต์เฉพาะในที่สาธารณะหากคุณกังวลเรื่องความเป็นส่วนตัว

#### วิธีการรายงาน:

1. ไปที่ส่วน [Issues](https://github.com/PhyschicWinter9/thai-adblock-list/issues) ของ repository นี้
2. คลิกที่ปุ่ม "New Issue"
3. ให้ข้อมูลที่ละเอียดที่สุดเท่าที่จะทำได้:
   - เบราว์เซอร์และเวอร์ชันของส่วนขยายบล็อกโฆษณา
   - ระบบปฏิบัติการ
   - ขั้นตอนในการทำซ้ำ
   - ภาพหน้าจอ (แนะนำ)
4. ชุมชนจะตรวจสอบและดำเนินการแก้ไขโดยเร็วที่สุด

#### หมายเหตุเรื่องความเป็นส่วนตัว:

หากคุณไม่สบายใจที่จะโพสต์ URL ของเว็บไซต์ในที่สาธารณะ คุณสามารถ:

- ส่งรายละเอียดผ่านข้อความส่วนตัวของ GitHub ถึงผู้ดูแล
- อธิบายประเภทโฆษณาและพฤติกรรมในแง่ทั่วไป
- ใช้ชื่อที่อธิบายแทน URL จริง (เช่น "เว็บไซต์สตรีมมิ่งภาษาไทยยอดนิยม")

ผู้ดูแลจะทำงานร่วมกับคุณเพื่อระบุและแก้ไขปัญหาโดยไม่จำเป็นต้องเปิดเผยเว็บไซต์เฉพาะในที่สาธารณะ
